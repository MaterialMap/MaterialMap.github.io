About Material MAP

Project Overview

Material MAP is a non-commercial, ad-free static website project designed to assist engineers in finding examples of material model parameter sets for LS-DYNA, sourced from articles published under Open Access. The project provides a searchable library of material models with direct links to original open-access sources and citation information.

Key Features

  • Searchable Database: Browse and filter through a comprehensive collection of LS-DYNA material models with advanced search capabilities
  • Material Calculators: Interactive calculators for various material models including:
    • Swift's Law Calculator for metal strain hardening
    • Mooney-Rivlin Calculator for hyperelastic materials
    • Johnson-Cook Calculator for rate-dependent plasticity
    • Gibson-Ashby Calculator for cellular materials
    • CEB-FIP Calculator for concrete properties based on Model Code 2010 and Eurocode 2
  • PWA Support: Progressive Web App functionality with offline capabilities through service worker caching
  • Open Access Sources: All material data references point to freely accessible open-access publications
  • Automatic Date Tracking: File modification dates are automatically tracked using Git history
  • Copy-Ready Format: Material data is stored as multi-line strings that can be copied directly to LS-DYNA input files

Technical Implementation

The website is a static site built from the MaterialMap GitHub repository. The entire database is stored in multiple TOML files in the data directory, allowing for easy updates and community contributions.

Data Structure

Each material entry includes:

  • Material Data: LS-DYNA material model definitions with proper formatting
  • EOS Data: Equation of state definitions when applicable
  • Additional Data: Thermal and supplementary material properties
  • Applications: List of material types and use cases
  • References: Full citations and links to original sources

Contributing

Community contributions are welcome and encouraged! You can contribute by:

  • Adding new material models through pull requests
  • Correcting errors in existing data
  • Reporting issues through GitHub Issues
  • Improving documentation and functionality

Requirements for contributions:

  • Source must be published under Open Access license
  • LS-DYNA material parameters must be validated
  • Reference URL must be accessible without paywall
  • All required TOML fields must be completed

Disclaimer

Important: All data provided on Material MAP is to be used at the user's own risk. While we strive for accuracy, users should always validate material parameters for their specific applications. The calculators are designed to provide numerically stable transitions and should not be considered as improvements to calculation accuracy without comprehensive validation.

Benefits

Material MAP creates a win-win situation: users benefit from a comprehensive materials library with ready-to-use LS-DYNA parameters, while authors of the referenced articles gain additional opportunities for increased citations and broader impact of their research.
