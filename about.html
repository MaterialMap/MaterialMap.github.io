About Materials MAP

This website and project are non-commercial and ad-free. The primary goal is to assist engineers in finding examples of material model parameter sets for LS-DYNA, sourced from articles published under Open Access. Accordingly, this site does not store any data and always provides links to original open-access sources, along with citation information for each reference.

The website is a static site built from the MaterialMap GitHub repository, and anyone is welcome to contribute by adding a new article with LS-DYNA material models to the site’s library or correcting errors through GitHub Issues or by editing the site's files. Currently, the entire site’s database is stored in multipes YAML files in data repository directory.

This project is licensed under the Creative Commons Attribution-NonCommercial (CC BY-NC) License, allowing non-commercial copying and modification with attribution to the original project.

In this way, users benefit from a kind of materials library, while authors of the referenced articles gain additional opportunities for increased citations. Please note that all data provided here is to be used at the user's own risk.
